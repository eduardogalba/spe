    API Reference spe v#1.0.0

    Modules

    Job

    Job module is responsible for managing the execution of tasks in a distributed system. It handles task assignment to workers, monitors their completion, and manages the overall state of the job. It uses a GenServer to maintain the state and handle asynchronous messages related to task execution and worker readiness.

    JobManager

    JobManager is responsible for managing jobs, including submitting jobs, starting them, and handling their plans. It uses a GenServer to maintain state and handle calls related to job management.

    Planner

    Planner is responsible for planning jobs based on their descriptions and the number of workers available. It implements Kahn's algorithm to create a plan for executing tasks while respecting their dependencies.

    SPE

    SPE is the main module for the SPE (Scalable Parallel Execution) system. It is responsible for starting the server, managing job submissions, and handling job starts. It uses a GenServer to maintain the state of the system and handle calls related to job management.

    SuperJob

    SuperJob is responsible for managing the lifecycle of a job, including starting the job and its associated workers. It uses a Supervisor to manage the job and its workers, ensuring that they are restarted if they fail.

    SuperManager

    SuperManager is responsible for managing the lifecycle of jobs in the SPE system. It uses a Supervisor to manage the SuperJob processes, which handle the execution of jobs and their associated workers.

    SuperSPE

    SuperSPE is the main supervisor for the SuperSPE application. It manages the SuperManager and JobManager processes.

    SuperWorker

    SuperWorker is responsible for managing a set of workers that execute jobs in parallel. It uses a Supervisor to manage the workers, ensuring that they are restarted if they fail.

    Validator

    Validator is responsible for validating job descriptions and tasks within the SPE system. It checks if the job description is a valid map, contains necessary fields, and ensures that tasks are properly defined.

    Worker

    Worker is responsible for executing tasks within a job in the SPE system. It is a GenServer that handles task execution, including applying functions with parameters and managing timeouts.
